# AGENTS.md

## Project scope

This repository contains the AI Tarot Companion web application.

## Development guidelines

- Use React and TypeScript.
- Use Tailwind CSS for styling.
- Keep reusable UI in `src/components` and route-level views in `src/pages`.
- Keep static domain data in `src/data`, shared helpers in `src/lib`, global
  styles in `src/styles`, and shared TypeScript definitions in `src/types`.
- Do not commit secrets, API keys, `.env` files, build output, or dependencies.
- Run `npm run build` and `npm run lint` before committing functional changes.
