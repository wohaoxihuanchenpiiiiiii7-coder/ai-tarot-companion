# AI Tarot Companion

Initial project scaffold for an AI-assisted tarot reading web application.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Local development

```bash
npm install
npm run dev
```

Open the local URL printed by Vite (usually `http://localhost:5173`).

## Production check

```bash
npm run build
npm run preview
```

## Environment variables

Copy `.env.example` to `.env` when environment variables are introduced. Never
commit `.env` files or API keys.
